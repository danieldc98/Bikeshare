Analyzing Chicago Bikeshare Data

Project Objectives

  • To learn about
    • -Relational databases and creating models from a dataset
    • -Django, a Python web framework, that uses a model-view-controller design pattern for data-driven websites
    • -Customizing web templates using HTML and CSS
  • Solve a problem and implement a solution using a dataset
  • Explore a dataset and implement a full-stack development environment (Databases, web framework, and front-end web design)
  • Report statistical findings on a modern, locally-hosted website

Approach

  • Download data for 368,371 bike trips as Comma-Separated-Values (csv)
  • Write Python script to convert dataset into Django models
  • Use Django, a high-level Python Web framework, to query and display data
